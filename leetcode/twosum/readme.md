## 两数之和 leetcode1
```bash
给定 nums = [2, 7, 11, 15], target = 9
因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]
```
- 暴力法 1.js  
  第一重循环计算target与nums[i]的差值，第二重循环匹配差值
  时间复杂度O(n^2)
  
- 优化 2.js  
  每遍历到一个元素就计算该元素与 target之间的差值 dif，然后以 dif 为下标到数组temp中寻找，如果 temp[dif] 有值，则返回两个元素在数组 nums 的下标，如果没有找到，则将当前元素存入数组 temptemp 中
